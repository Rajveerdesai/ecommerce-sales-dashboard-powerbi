📊 E-Commerce Sales Dashboard (Power BI)
🔹 Project Overview
This project presents an interactive E-Commerce Sales Dashboard built using Power BI to analyze sales performance, customer behavior, and business insights.

🔹 Key Features
Sales trend analysis (monthly & yearly)
Top-selling products identification
Customer segmentation
Region-wise sales performance
Profit and revenue insights
🔹 Tools & Technologies Used
Power BI
Microsoft Excel / CSV
Data Cleaning & Transformation
DAX (Data Analysis Expressions)
🔹 Dashboard Preview
🔹 Insights Generated
Identified highest revenue-generating products
Analyzed seasonal sales trends
Found top-performing regions
Improved decision-making using data visualization
